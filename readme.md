# setup mac
- set up mac using ansible

## how to use

- install homebrew 
    ```
    $ /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install.sh)"
    ```

- install ansible
    ```
    $ brew install ansible
    ```

- run ansible playbook
    ```
    $ ansible-playbook ./ansible/site.yml
    ```
